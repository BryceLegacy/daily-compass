# Security Policy

Daily Compass is a prompt-and-template repo. The main security risk is accidental disclosure of private meeting data or local context.

## Do Not Share In Issues Or Pull Requests

- real meeting notes
- transcripts
- private exports
- emails
- credentials or tokens
- private filesystem paths
- client names
- proprietary project details
- screenshots of private systems

## Reporting A Problem

If you find sensitive data in this public repo, open an issue that describes the file and line without repeating the sensitive value.

If the disclosure is severe, contact the repository owner privately through GitHub before opening a public issue.

## Supported Versions

This template is maintained on the `main` branch.
